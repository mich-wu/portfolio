yarn install

## Running locally

Setup .env file where applicable, contact a developer to get this information
.env.development for local live server

Enter command:
yarn develop

if you've set up everything right your local server will now be available on localhost:8000

Some projects may have the option to build locally using yarn build (check you package.json)
For this you will require the .env.production file

## Bash commands

Create a new component:

1. bash component.sh
2. enter component name using PascalCase convention
3. Begin coding!

Create a new content block:

1. bash block.sh
2. enter block name using PascalCase convention
3. Import GraphQL query fragments to all relevant page templates
4. Begin coding
